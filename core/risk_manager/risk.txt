
交易标的
# 1. 股票 (Equities)
# 上市公司股票：包括主要股票市场的蓝筹股、中型股、小型股，以及新兴市场股票。
# 股票衍生品：如股票期权、股票期货和差价合约 (CFDs)。
# 股指期货和期权：如标普500期货、纳斯达克100期权等。
# 2. 债券和固定收益 (Bonds and Fixed-Income)
# 政府债券：如美国国债、欧洲主权债券及其他国家的国债。
# 公司债券：企业发行的债券，包括投资级和高收益（垃圾债）。
# 地方政府债券（市政债券）：州和当地政府发行的债券。
# **资产支持证券 (ABS) 和抵押贷款支持证券 (MBS)**。
# 3. 外汇 (Forex)
# 主要货币对：如美元/欧元（EUR/USD）、美元/日元（USD/JPY）、英镑/美元（GBP/USD）等。
# 新兴市场货币对：涉及新兴市场国家的货币。
# 4. 大宗商品 (Commodities)
# 能源商品：如原油（WTI、Brent）、天然气等。
# 金属：如黄金、白银、铜、铝等。
# 农业商品：如小麦、玉米、大豆等。
# 软商品：如咖啡、棉花、糖等。
# 5. 对冲基金产品 (Hedge Funds)
# 多空股票策略：通过做多和做空股票市场的份额来管理风险。
# 事件驱动策略：基于公司事件（如并购、重组）进行交易。
# 全球宏观策略：根据宏观经济趋势和政策变化进行跨市场和跨资产的交易。
# 6. 私募股权投资 (Private Equity)
# 直接股权投资：对未上市企业进行股权投资。
# 杠杆收购（LBOs）：利用杠杆进行企业收购。
# 风险投资（VC）：投资初创企业和高增长潜力企业。
# 7. 房地产投资 (Real Estate)
# 商用和住宅地产投资信托（REITs）：投资房地产相关的证券。
# 直接房地产投资：购买和管理实际房地产资产。
# 8. 结构性产品 (Structured Products)
# 结构性票据：根据基础资产（如股票、债券、商品等）设计的定制化投资工具。
# **信用违约互换(CDS)**：用于对冲债务违约风险的金融衍生产品。
# **资产组合保险 (Portfolio Insurance)**：设计对某些市场风险自动对冲的产品。
# 9. 其他替代投资 (Alternative Investments)
# 加密货币：如比特币、以太坊及其他区块链相关资产（视机构政策和监管情况而定）。

# 交易策略风控
# 1、成交量指标：买入时，交易品种成交量不得超过一段时期65%百分位数交易量；
# 2、"超买" / "超卖" 指标：买入时：买入时，市场不能处于短期超买状态（最近7个交易日）；


# 交易成本
# 1、每次短期买入，不能连续在15天内执行；
# 2、每次长期买入，不能连续在30天内执行；
# 3、买入交易当天收益率，标地价格涨幅不能超过0.5%；



# 仓位风控
# 头寸限额：限制每个资产或每类资产的最大持仓量，不能超过最大持仓量
# 杠杆比率：控制整体投资组合的杠杆水平。
# 最大回撤：设定账户净值的最大允许回撤额度。
# 单笔交易限额：限制单笔交易的风险敞口。
